# 콘텐츠 인덱스

> AI가 콘텐츠를 추가/수정할 때 이 파일을 반드시 업데이트한다.
> 이 인덱스는 블로그 랜딩 페이지(index.html) 생성 시 데이터 소스로 사용된다.

## 시리즈 목록

유튜브 영상 "99%가 인생을 낭비하는 이유"에서 출발한 5편 시리즈. 원본 정리 → 반박 → 종합 → 실행 → 메타 순서.

| # | 슬러그 | 한글 제목 | 역할 | 생성일 |
|---|--------|-----------|------|--------|

| 02 | `what-self-help-wont-tell-you` | 자기계발이 말하지 않는 것 | 팩트 기반 반박 (학술 논문/OECD 데이터 인용) | 2026-02-08 |
| 03 | `conditions-for-change` | 변화의 조건 | 종합 분석 — 양쪽 맹점 교차 검증 | 2026-02-08 |
| 04 | `so-what-now` | 그래서 어떡하라고 | 실행 매뉴얼 — 자가 진단 + 90일 프로토콜 | 2026-02-08 |

### bithumb-60t-series — "딸깍 한 번의 60조 원" 시리즈

SBS 뉴스 "60조 원 '딸깍' 한 번의 대참사"에서 출발한 시리즈. 빗썸 비트코인 오지급 사고의 전말과 가상자산 거래소의 구조적 문제를 분석한다.

| # | 슬러그 | 한글 제목 | 역할 | 생성일 |
|---|--------|-----------|------|--------|
| 01 | `bithumb-60-trillion-incident` | 딸깍 한 번의 60조 원 | 원본 정리 — SBS 뉴스 + 자료조사 기반 사고 분석 에디토리얼 | 2026-02-08 |
| 02 | `whose-money-is-it` | 그 돈은 누구의 것인가 | 법적 분석 — 수익화 경로, 판례, 시나리오 검증 | 2026-02-08 |

## 인덱스 관리 규칙

1. **새 콘텐츠 추가 시** 해당 시리즈 테이블에 행을 추가한다
2. **새 시리즈 추가 시** `### 시리즈-슬러그 — "시리즈 제목"` 형식으로 섹션을 추가한다
3. 슬러그는 영문 소문자 + 하이픈만 사용한다
4. 파일 경로는 `content/[시리즈-슬러그]/[파일-슬러그].html`이다
