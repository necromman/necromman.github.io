# 콘텐츠 인덱스

> AI가 콘텐츠를 추가/수정할 때 이 파일을 반드시 업데이트한다.
> 랜딩 페이지의 실제 런타임 데이터는 `assets/content-data.js`에서 관리한다. 이 파일은 사람/AI가 콘텐츠 현황을 빠르게 파악하기 위한 기록용이다.

## 인덱스 관리 규칙

1. **새 콘텐츠 추가 시** 해당 시리즈 테이블에 행을 추가한다
2. **새 시리즈 추가 시** `### 시리즈-슬러그 — "시리즈 제목"` 형식으로 섹션을 추가한다
3. 슬러그는 영문 소문자 + 하이픈만 사용한다
4. 파일 경로는 `content/[시리즈-슬러그]/[파일-슬러그].html`이다
