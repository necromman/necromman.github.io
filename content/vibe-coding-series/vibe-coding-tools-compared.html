Part I

도구의 두 세계

AI 코딩 도구를 하나로 묶어서 비교하는 것은 "자동차와 택시 중 뭐가 나아?"라고 묻는 것과 같다. 직접 운전하느냐, 목적지만 말하느냐의 차이다. AI 코딩 도구도 크게 두 세계로 나뉜다.

Category A

IDE/에디터형

기존 코드 에디터에 AI를 탑재한 도구. 코드를 읽을 줄 아는 사람이 대상이다. Cursor, GitHub Copilot, Windsurf, Claude Code가 여기에 속한다.

Category B

앱 빌더형

코드를 보여주지 않고 결과물만 보여주는 도구. 비개발자도 쓸 수 있다. Bolt.new, Lovable, Replit, v0가 여기에 속한다.

어떤 세계의 도구를 쓸지는 당신이 누구냐에 달려 있다. 코드를 읽을 줄 아는 개발자라면 IDE형 도구가 훨씬 강력하다. 프로그래밍 경험이 없지만 아이디어를 앱으로 만들고 싶은 사람이라면 앱 빌더형이 맞다. 두 세계를 따로 뜯어보자.

Part II

IDE형 4종 비교

IDE형 도구는 모두 VS Code라는 코드 에디터를 기반으로 한다. 마이크로소프트가 만든 무료 에디터인데, 전 세계 개발자의 73%가 사용한다. 이 에디터에 AI 능력을 얹는 방식이 도구마다 다르다.

IDE Type

Cursor

무료 / $20 / $60 / $200/월

VS Code를 통째로 복제(fork)해서 AI를 심층 통합한 에디터. Agent 모드로 여러 파일을 동시에 수정하고, 프로젝트 전체 맥락을 이해한다. 일일 활성 사용자 100만+.

개인 개발자, 소규모 팀. "AI 코딩의 체감 속도"를 원한다면.
IDE Type

GitHub Copilot

무료 / $10 / $39/월

마이크로소프트가 만든 원조 AI 코딩 도구. VS Code, JetBrains, Neovim 등 거의 모든 에디터에서 작동한다. 전체 사용자 2,000만+. GPT-5, Claude Opus 4 등 최신 모델 지원.

범용 개발자, 이미 VS Code를 쓰는 사람. "가장 안전한 선택".
IDE Type

Windsurf

무료 / $15 / $30/월

Codeium에서 만든 AI IDE. Cascade라는 에이전트가 여러 단계의 작업을 계획하고 실행한다. 엔터프라이즈급 보안(Zero-Data Sovereignty)이 강점. 코드가 로컬을 벗어나지 않는 옵션 제공.

보안이 중요한 기업. "데이터가 밖으로 나가면 안 되는" 환경.
CLI Type

Claude Code

$20 ~ $200/월 (Max)

Anthropic이 만든 터미널 기반 AI 에이전트. GUI 에디터가 아니라 터미널에서 대화하며 코딩한다. 벤치마크 정확도 80.9%로 최고 수준. 다른 도구 대비 코드 재작업 30% 적음.

시니어 개발자. "정확도와 깊이 있는 분석"을 원한다면.

Cursor는 속도, Copilot은 범용성, Windsurf는 보안, Claude Code는 정확도.

Part III

앱 빌더형 4종 비교

앱 빌더형 도구는 코드를 전혀 몰라도 작동하는 앱을 만들 수 있게 해준다. 채팅으로 원하는 걸 설명하면 AI가 코드를 짜고, 화면을 그리고, 심지어 배포까지 해준다. 다만 각 도구의 철학과 강점이 다르다.

App Builder

Bolt.new

무료 / 오픈소스

브라우저 안에서 모든 것이 돌아간다. WebContainer 기술로 밀리초 만에 개발 환경이 부팅된다. React, Vue, Svelte 등 프레임워크 선택의 자유가 가장 넓다. 오픈소스라 커스텀도 가능.

빠른 프로토타이핑. "5분 안에 뭔가 돌아가는 걸 보고 싶다면".
App Builder

Lovable

무료 / $25~39/월

채팅으로 대화하면 풀스택 앱이 만들어진다. "시니어 개발자와 대화하는 느낌"이라는 평가가 많다. Supabase(데이터베이스)와 Stripe(결제) 통합이 매끄러워서 실제 수익을 내는 앱까지 가능.

비기술 창업자. "아이디어를 몇 시간 만에 MVP로 만들고 싶다면".
App Builder

Replit

무료 / $25/월

클라우드 기반 IDE + AI Agent. 30개 이상의 외부 서비스와 통합되고, 코드 작성부터 배포까지 한 곳에서 끝난다. 3,000만 명의 커뮤니티. 학습용으로도 최적.

입문자, 학생. "코딩을 배우면서 동시에 만들고 싶다면".
App Builder

v0 (Vercel)

무료 / $39/월

Next.js에 특화된 AI 앱 빌더. UI가 가장 아름답다는 평가를 받는다. 빌트인 데이터베이스를 제공하고, 배포까지의 흐름이 가장 매끄럽다. 다만 Next.js 생태계에 묶인다.

디자인 감각이 있는 개발자. "예쁜 웹앱을 빠르게 만들고 싶다면".
Part IV

"나는 ___인데, 뭘 써야 해?"

도구의 스펙을 외워도 결국 선택은 "나는 누구고, 뭘 만들려는가"에 달려 있다. 아래 의사결정 가이드를 따라가면 된다.

의사결정 가이드
  • 프로그래밍 경험이 전혀 없고, 앱 아이디어를 구현하고 싶다 Lovable / Replit
  • 코딩을 배우면서 동시에 프로젝트를 만들고 싶다 Replit
  • 5분 안에 프로토타입을 보고 싶고, 돈은 안 쓰고 싶다 Bolt.new
  • Next.js로 예쁜 웹사이트를 빠르게 만들고 싶다 v0
  • 주니어~미드 개발자이고, AI로 생산성을 올리고 싶다 Cursor / Copilot
  • 이미 VS Code를 쓰고 있고, 가장 무난한 선택을 원한다 GitHub Copilot
  • 멀티파일 리팩토링, 복잡한 프로젝트를 AI에게 맡기고 싶다 Cursor
  • 시니어 개발자이고, 터미널에서 정확한 코드를 원한다 Claude Code
  • 기업 환경이고, 코드가 외부로 나가면 안 된다 Windsurf
Part V

가격의 진실

"무료"라는 말에 속으면 안 된다. 대부분의 AI 코딩 도구는 무료 티어의 제한이 엄격하다. 실제로 일하려면 유료 구독이 필요하고, 일부 도구는 사용량에 따라 추가 요금이 붙는다.

도구 무료 유료 시작 숨겨진 비용
Cursor 50회/월 제한 $20/월 $60 Pro+ 안 쓰면 Agent 기능 제한
Copilot 월 한도 있음 $10/월 $39 Pro+ 안 쓰면 최신 모델 제한
Windsurf 25 크레딧/월 $15/월 크레딧 소진 시 추가 구매 필요
Claude Code 없음 $20/월 토큰 사용량 기반, 과금 예측 어려움
Bolt.new 있음 (오픈소스) 무료 호스팅/배포는 별도
Lovable 있음 (제한적) $25/월 Supabase, Stripe 등 외부 서비스 비용
Replit 있음 $25/월 컴퓨팅 크레딧 소진 시 추가 비용
v0 있음 (제한적) $39/월 Vercel 호스팅 비용 별도

2025년 6월, Cursor는 가격 체계를 전면 개편했다. 고정 월정액에서 "기본 크레딧 + 초과 사용 과금" 모델로 바꿨다. 이 변화는 업계 전체의 방향을 보여준다. AI 모델을 돌리는 비용이 만만치 않기 때문에, "무제한"이라고 쓰고 "제한적"으로 읽어야 하는 시대가 됐다.

비용 관련 주의사항
  1. 무료 티어로 충분한지 반드시 먼저 테스트한다. 대부분 2~3일이면 한도에 도달한다.
  2. API 기반 도구(Claude Code 등)는 월별 사용량 알림을 설정한다. 프롬프트 한 줄 수정으로 11일 만에 $11,000가 청구된 사례가 있다.
  3. "월 $20"은 도구 비용만이다. 배포, 데이터베이스, 도메인 등 인프라 비용은 별도다.
  4. 연간 결제 시 20% 할인이 일반적이지만, 도구를 바꾸고 싶을 때 묶이는 리스크가 있다.
