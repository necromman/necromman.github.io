# 자료조사 — "강해져버린 이유"

## 원작 정보

- 원작: ONE (웹코믹 → 무라타 유스케 리메이크 만화)
- 장르: 액션, 코미디, 패러디
- 핵심: 모든 적을 한 방에 이기는 히어로의 일상과 권태

## 사이타마의 힘 — 원작에서 밝혀진 것

### Dr. Genus의 리미터 이론
- 모든 생물에는 "리미터(성장 상한)"가 있음
- 신(God)이 정한 한계. 초과하면 괴수화되거나 소멸
- 사이타마는 리미터를 **인간인 채로 깨뜨림** — 전례 없음
- 물리적 훈련이 아닌 **"절대 포기하지 않는 의지"**가 핵심

### 공식 훈련법
- 100번 팔굽혀펴기, 100번 윗몸일으키기, 100번 스쿼트, 10km 러닝
- 매일 3년간. 에어컨/난방 금지. 아침은 바나나 1개.
- 작중 모든 캐릭터가 "그걸로 이렇게 강해진다고?" 반응
- 핵심: 훈련 자체는 평범. 평범한 훈련으로 비범한 결과를 낸 게 역설.

### "God" 엔티티
- 차원 너머에 봉인된 러브크래프트적 우주 존재
- 미스터리어스 큐브를 통해 소통. 욕망을 말하면 힘을 부여(또는 소멸)
- 사이타마를 **"신을 거스른 주먹(the fist that turned against God)"**이라 부름
- 최종 빌런으로 설정됨 (만화 기준)

### 작가 ONE의 입장
- 사이타마의 힘 기원에 대해 **의도적으로 모호하게** 유지
- "이미 최강인 히어로의 일상"이 관심사. 파워업 서사가 아님.
- 설명이 불만족스러운 게 포인트 — 쇼넨 장르 패러디

### 가로우 전투에서 드러난 사실 (만화)
- 사이타마는 감정에 자극받으면 **기하급수적으로 성장**
- 가로우가 사이타마 힘을 복사했지만, 사이타마의 성장 속도를 따라잡지 못함
- 사이타마가 시간여행(시간을 주먹으로 되돌림) — 시공간 조작 능력 확인
- 이는 "단순한 힘"이 아니라 "차원적 존재"에 가까움을 시사

## 팬픽 설계 포인트

1. **사이타마가 자발적으로 퀘스트를 시작할 동기**: 힘의 비밀이 아닌 **탈모의 원인**
2. **미스터리어스 큐브를 따라가는 모험**: 큐브마다 진실의 파편 → 원피스식 여정
3. **블라스트의 차원 여행 능력**: 다른 차원으로의 이동 수단
4. **God과의 대면**: 최종 목적지. 단, 사이타마는 관심 없음.
5. **결론은 안티클라이맥스**: 원펀맨답게. 우주적 진실을 발견하지만 "그래서?" 반응.

## 톤/유머 규칙

- 우주적 진실 ↔ 사이타마의 일상적 관심사 (세일, 밥, 게임) 대비가 핵심
- 제노스의 과잉 분석 + 킹의 허세(실은 공포) = 동행 코미디
- 긴 대사/연설은 반드시 사이타마에 의해 잘림
- 전투는 한 방이지만, 차원이 다르면 물리법칙이 달라서 좀 더 걸릴 수 있음
